Generate simple libEnsemble scripts from a web form entry.

Try out at: https://libensemble.github.io/script-creator/

## Using with Cursor IDE (MCP)

You can also use this tool directly in Cursor IDE through the Model Context Protocol (MCP). See [MCP_SETUP.md](MCP_SETUP.md) for simple setup instructions.

## Running the agentic workflow scripts

You can run an agent that generates scripts and fixes up scripts on failures.
See [AGENTIC_SETUP.md](AGENTIC_SETUP.md) for instructions.
